My goal is to apply all the knowledgement I've got to program one game. Pou was chosen because it's a nostalgic and fun game. Also it doesn't seem to be so hard to recreate.

2. Game/Program Description

The game's goal is to take care of an alien, feed it, play with it, and also dress it as you want. The genre is virtual pet game, the target audience is kids and teens.

3. Variables

  • ClouthCode -> Armazena a posição em que está armazenada a imagem da roupa atual do personagem.
  • Clouths -> É um array que armazena todas as possibilidades de roupa do personagem.
  • SkinColours -> É um array que armazena todas as possibilidades de tons de pele do personagem.
  • SkinColourNow -> Armazena um inteiro que diz respeito à cor atual do personagem.

4. Functions

  • abrirMenu -> abre o menu do jogo
  • status -> controla os status de fome/diversão/limpeza/energia e informa
    • -> abre o seguimento no qual é possivel trocar a roupa do personagem
  • inventario -> mostra a quantidade de moedas na tela
