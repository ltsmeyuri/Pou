1. Introduction

My goal is to apply all the knowledgement I've got to program one game. Pou was chosen because it's a nostalgic and fun game. Also it doesn't seem to be so hard to recreate.

2. Game/Program Description

The game's goal is to take care of an alien, feed it, play with it, and also dress it as you want. The genre is virtual pet game, the target audience is kids and teens.

3. Variables

  • ClothCode -> Stores the code (the number in the clothes array) that represents the clothes the character is currently wearing.
  • Cloths -> It's an array that stores all the clothes available in the game.
  • SkinColours -> It's an array that contains all the possible skin colours in it.
  • SkinColourNow -> An integer value that represents the character's skin colour.

4. Functions

  • openMenu -> open the game's menu
  • showStatus -> sets the hunger, fun, cleanliness, and energy statuses and shows them on the screen.
  • openBag -> Shows the amount of money.
